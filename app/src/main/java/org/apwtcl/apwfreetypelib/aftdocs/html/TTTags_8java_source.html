197  * Used by Apple to indicate a MacOS-specific charmap and/or name entry.
198  * See @TT_MAC_ID_XXX for corresponding `encoding_id' values. Note that
199  * most TrueType fonts contain an Apple roman charmap to be usable on
200  * MacOS systems (even if they contain a Microsoft charmap as well).
201  *
202  * TT_PLATFORM_ISO ::
203  * This value was used to specify ISO/IEC 10646 charmaps. It is however
204  * now deprecated. See @TT_ISO_ID_XXX for a list of corresponding
205  * `encoding_id' values.
206  *
207  * TT_PLATFORM_MICROSOFT ::
208  * Used by Microsoft to indicate Windows-specific charmaps. See
209  * @TT_MS_ID_XXX for a list of corresponding `encoding_id' values.
210  * Note that most fonts contain a Unicode charmap using
211  * (TT_PLATFORM_MICROSOFT, @TT_MS_ID_UNICODE_CS).
212  *
213  * TT_PLATFORM_CUSTOM ::
214  * Used to indicate application-specific charmaps.
215  *
216  * TT_PLATFORM_ADOBE ::
217  * This value isn't part of any font format specification, but is used
218  * by FreeType to report Adobe-specific charmaps in an @FT_CharMapRec
219  * structure. See @TT_ADOBE_ID_XXX.
